# About Me

I am an Artificial Intelligence student and developer focused on building practical AI systems, automation tools, machine learning projects, and intelligent agents.

My work is centered around building complete systems, not just isolated scripts. I like connecting AI with real data, real workflows, databases, APIs, dashboards, and user-facing applications.

I am interested in AI agents, RAG systems, LLM applications, machine learning, automation, data analysis, and full-stack AI products.
